# LELAPE_jll
A Julia package to classify single event upsets in memories an FPGA into events of different multiplicity.  It can be used in combination with the [LELAPE](https://github.com/fjfrancopelaez/LELAPE) software.

## How to install it
Open the Julia REPL and type:

    using Pkg;
    Pkg.add(url="https://github.com/fjfrancopelaez/LELAPE_jll.git");

and finally type 

    using LELAPE_jll

to load the toolbox.
